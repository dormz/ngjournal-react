# Ant Design Component Semantic Description

This document contains semantic descriptions of all components in the Ant Design component library.

A total of 65 components contain semantic descriptions.

## Component List

### alert

- `root`: The root element, containing the basic styles of the warning box, including border, background color, padding, rounded corners, and position.
- `section`: Content element, using flexbox layout to control the layout and minimum width of the content area.
- `icon`: Icon element, including icon color, line height, margin, and other styles, supporting different types of status icons.
- `title`: The title element, containing the title text's color, font, and other styles.
- `description`: Description element, containing the font size, line height, and other layout styles of the descriptive text.
- `actions`: Action group elements, containing the layout and spacing style of the action buttons.
- `close`: Closes the button element, including the button's basic styles.

### anchor

- `root`: The root element, containing basic styles such as layout positioning, padding, margins, and background color.
- `item`: Link item element, including styles such as padding, text color, hover state, and transition animation.
- `itemTitle`: Title text element, including font style, color variations, text decorations, transition effects, and other styles.
- `indicator`: An indicator element that includes styles for width, height, background color, position change, and transition animations.

### badge

- `root`: The root element, containing basic layout styles such as relative positioning, inline-block layout, and adapting to content width.
- `indicator`: An indicator element that includes complete logo styles such as positioning, hierarchy, size, color, font, text alignment, background, rounded corners, shadow, and transition animations.

### breadcrumb

- `root`: The root element, containing basic styles such as text color, font size, and icon size. Internally, it's an ordered list using flexbox layout.
- `item`: The Item element, which includes styles such as text color, link color variation, hover effect, padding, rounded corners, height, and margin.
- `separator`: Separator element, containing the separator's margin and color style.

### button

- `root`: The root element, containing complete button appearance styles including border style, background color, padding, rounded corners, shadow effects, transition animations, cursor style, text weight, alignment, etc.
- `content`: The content element, which wraps the button text content and controls text layout styles such as text non-wrapping, center alignment, and optimized spacing of Chinese characters.
- `icon`: Icon element, including icon display-related styles such as font size, color inheritance, and SVG style reset.

### calendar

- `root`: The root element, containing the calendar component's basic styles such as background color, border, and rounded corners, as well as its overall layout structure.
- `header`: The header element, containing layout and style controls for the year selector, month selector, and mode switcher.
- `body`: The main element, containing the calendar table's padding, layout controls, and other styles, used to house the calendar grid.
- `content`: The content element, containing the size controls for the calendar table, such as its width and height, as well as its table style.
- `item`: The item element, containing the calendar cell's background color, border, hover state, selected state, and other interactive styles.

### card

- `root`: The root element of the card container, containing basic styles such as positioning, background color, border, rounded corners, shadow, and padding.
- `header`: The header area of ​​the card, containing styles such as flex layout, minimum height, padding, text color, font weight, font size, background color, bottom border, and top rounded corners.
- `body`: The card's content area, including basic styles for content display such as padding and font size.
- `extra`: The operation area in the upper right corner of the card, containing the text color and layout style of additional content.
- `title`: Card title, including inline-block layout, flex proportions, text omission, and other title display styles.
- `actions`: The action group at the bottom of the card, including flex layout, list style reset, background color, top border, bottom rounded corners, and other action button container styles.
- `cover`: Title/cover page, including the display and layout style of the cover image.

### card:meta

- `root`: Sets the root element of metadata.
- `section`: Sets the metadata content element
- `avatar`: Sets the metadata icon
- `title`: Set the metadata title
- `description`: Sets the metadata description.

### checkbox

- `root`: The root element, containing the basic styles of the checkbox container, including inline flex layout, baseline alignment, cursor style, reset style, etc.
- `icon`: A checkbox element, including size, orientation, background color, border, rounded corners, transition animation, and the checkmark style for the selected state.
- `label`: A text element that includes the padding of the text and the spacing style between it and the checkbox.

### collapse

- `root`: The root element, containing container styles such as the border, rounded corners, and background color of the collapsible panel, and defining the overall layout and appearance of the control panel.
- `header`: The header element, containing flex layout, padding, color, line height, cursor style, transition animations, and other interactive and stylistic elements of the panel header.
- `title`: The title element, including the layout and typography style of the title text, such as flexbox adaptive layout and right margin.
- `body`: Content element, including padding, color, background color, and other display styles for the panel content area.
- `icon`: Icon elements, including font size, transition animations, rotation transformations, and styles and effects for expanding and collapsing arrows.

### colorPicker

- `root`: The trigger container, containing border styles, transition animations, size controls, and displaying color blocks and text content.
- `popup`:
  - `root`: The root container of the pop-up panel, containing styles such as background color, shadow effect, color selection panel, slider controls, and preset colors.

### datePicker

- `root`: The root element, containing the basic styles of the date picker container, including relative positioning, inline flex layout, padding, rounded corners, and transition animations.
- `prefix`: Prefix element, containing layout styles such as flex layout and right margin.
- `input`: The input box element, containing the core interactive styles of the input box, such as relative positioning, width, color, font, line height, and transition animations.
- `suffix`: A suffix element containing styles for flex layout, color, line height, pointer events, transition animations, etc.
- `popup`:
  - `container`: The container element, which sets the background color, padding, rounded corners, shadow, border, and content display style.
  - `header`: The header element of the popup, containing the layout and style of the header control area, including navigation buttons, month and year pickers, etc.
  - `body`: The main element of the popup, containing the container layout and style of the date panel table.
  - `content`: The pop-up content element, including the width, borders, cell layout, and display style of the date table.
  - `item`: A pop-up item element containing interactive styles such as date cell size, background color, rounded corners, hover state, and selected state.
  - `footer`: The element at the bottom of the pop-up, containing the layout style of the bottom operation area such as confirm/cancel buttons and quick selection options.

### descriptions

- `root`: The root element, containing the overall styles of the list container, including its base styles, reset styles, border styles, and layout direction.
- `header`: The header element, containing layout and style controls for the header area, including flexbox layout, alignment, bottom margin, etc.
- `title`: A heading element that includes heading text styles such as text omission, flex proportion, color, font weight, font size, and line height.
- `extra`: Additional content elements, including styles for extra operational areas such as left margin, color, and font size.
- `label`: A label element that includes styles for the label text, such as color, font weight, font size, line height, text alignment, and colon style.
- `content`: Content element, including table cell layout, color, font size, row height, text wrapping, and other content display styles.

### divider

- `root`: The root element, containing the basic styles of the dividing container, including border and top styles, and separator styles.
- `content`: The content element, containing styles for inline-block text, padding, and other separator text content.
- `rail`: Background bar element, including styles for borders, top styles, and connecting lines.

### drawer

- `root`: The root element, containing the basic styles and layout controls for the drawer container, including fixed positioning, hierarchy control, pointer events, and colors.
- `mask`: A masking element containing styles and interactive controls for the mask, including absolute positioning, hierarchy, background color, and pointer events.
- `section`: Drawer container element, containing the drawer's main body style, including flex layout, width and height, overflow control, background color, pointer events, etc.
- `header`: The header element, containing styles for the header area such as flex layout, alignment, padding, font size, line height, and bottom border.
- `body`: Content element, including the display and layout style of the content area, such as flex percentage, minimum size, padding, and overflow scrolling.
- `footer`: The footer element, containing styles for the bottom operational area, including flex shrinkage, padding, top border, etc.
- `title`: The title element, containing styles for the title text such as flex proportion, margin, font weight, font size, and line height.
- `extra`: Style controls for additional elements, including those with flexbox fixed layout and other extra operations.
- `dragger`: A draggable element with drag handles for resizing the drawer, including absolute positioning, transparent background, pointer event control, hover state styles, and drag state styles.
- `close`: Closes the button element, including the button's basic styles.

### dropdown

- `root`: The root element of the dropdown menu, used to set its positioning, hierarchy, and container styles.
- `itemTitle`: The title content area for dropdown options, setting layout and text style.
- `item`: A single option element in a dropdown menu, setting the option's interactive state and background style.
- `itemContent`: The main content area for dropdown options, setting content layout and link styles.
- `itemIcon`: The icon area for the dropdown options, setting the icon size and spacing style.

### empty

- `root`: The root element, used to set text alignment, font, and line height styles.
- `image`: Icon element, sets its height, transparency, margins, and image style.
- `description`: Describes the element and sets its text color and style.
- `footer`: The bottom element, used to set the top margin and style the action buttons.

### floatButton

- `root`: The root element, which sets the basic style, shape, size, type, theme, fixed positioning, hierarchy, shadow, spacing, and other container styles for the floating button.
- `content`: The content element, which sets the font size, color, alignment, line breaks, and other text display styles of the text content within the button.
- `icon`: Icon element, sets the size, color, line height, alignment, and other display styles of the icon within the button.

### floatButtonGroup

- `root`: The root element, which sets the container style, fixed positioning, hierarchy, padding, spacing, orientation mode, and other combined layout styles for the floating button group.
- `list`: A list element that sets the Flex layout, rounded corners, shadow, animation transitions, vertical alignment, and other list container styles for the button group list.
- `item`: List item element, sets the basic button style such as style, size, shape, type, state, icon content, etc. for a single floating button.
- `itemIcon`: List item icon element, sets the size, color, alignment, and other display styles of the icon within the floating button.
- `itemContent`: List item content element, sets the style of the text content, logo, description, and other content areas within the floating button.
- `trigger`: The trigger element that sets the style, shape, icon, hover state, expanded/collapse state, and other interactive styles of the trigger button in menu mode.
- `triggerIcon`: Trigger icon element, sets the style, rotation animation, state toggling, and other icon interaction styles of the icon inside the trigger button.
- `triggerContent`: Trigger content element, sets the style of text, logo, status indicator, and other content in the trigger button's content area.

### form

- `root`: The root element, containing the basic styles of the form item container, including bottom margin, vertical alignment, transition animations, hidden state, error warning state, etc.
- `label`: A label element that includes flex layout, overflow hiding, text non-wrapping, text alignment, vertical alignment, and label display styles such as label color, font size, height, and required field marking.
- `content`: The content element, containing the layout, style, and related styles of the form content area and control container.

### image

- `root`: The root element, used to set relative positioning and inline-block layout styles.
- `image`: Image element, sets width, height, and vertical alignment styles.
- `cover`: A tooltip element for displaying a floating image; its absolute positioning, background color, transparency, and transition animation style can be set.
- `popup`:
  - `root`: Preview the root element, set its fixed positioning, hierarchy, and background mask style.
  - `mask`: Preview the masking element, set absolute positioning and semi-transparent background style.
  - `body`: Preview content elements, set flex layout, center alignment, and pointer event styles.
  - `footer`: Preview the footer element, set its absolute positioning, center alignment, and style for the bottom navigation area.
  - `actions`: Preview the action group elements, set the flex layout, background color, rounded corners, and action button styles.

### input

- `root`: The root element, containing the basic styles of the input box container, including relative positioning, inline-block layout, width, minimum width, padding, color, font, line height, rounded corners, and transition animations.
- `input`: The input box element, containing the core interactive styles and text input-related styles.
- `prefix`: The wrapper element for the prefix, containing the layout and styles of the prefix content.
- `suffix`: The wrapper element for the suffix, containing the layout and styles of the suffix content.
- `count`: A character count element that includes the font and color style for displaying the character count.

### input:password

- `suffix`: The wrapper element with the suffix, which sets the flex layout, margins, and transition animation styles.
- `action`: A single action button element that sets the button's style, hover effect, and click interaction.
- `actions`: Manipulates elements, setting absolute positioning, width, flex layout, and style of numeric adjustment buttons.

### inputSearch

### list

### masonry

### mentions

### menu

- `root`: The root element, containing the basic styles and layout of the menu container.
- `item`: The item element, containing basic interactive styles for menu items such as relative positioning, block-level display, margins, whitespace handling, cursor styles, and transition animations.
- `itemContent`: The item content element, containing the layout and typography style of the menu item content.
- `itemIcon`: Icon element, including minimum width, font size, transition animation, icon reset style, and spacing control between the icon and text.
- `itemTitle`: Menu title element (not applicable in horizontal mode), containing the style and layout of the title text.
- `list`: Menu list element (not applicable in horizontal mode), including the layout and container style of the menu list.
- `popup`: Popup menu (does not work in inline mode), including popup positioning, hierarchy, background, and other styles.
- `subMenu`:
  - `itemTitle`: Submenu title element, containing the style and interactive effects of the submenu title.
  - `list`: A submenu list element that includes the layout and container style for the submenu list.
  - `item`: A submenu item element that includes the style and interactive effects of the submenu item.
  - `itemIcon`: Submenu item icon element, containing the size and style of the submenu icon.
  - `itemContent`: The submenu item content element, containing the layout and typography of the submenu content.
